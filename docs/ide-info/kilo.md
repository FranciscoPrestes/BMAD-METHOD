# BEAT Method - KiloCode Instructions

## Activating Agents

BEAT agents are installed as custom modes in `.kilocodemodes`.

### How to Use

1. **Open Project**: Modes auto-load when project opens
2. **Select Mode**: Use mode selector in KiloCode interface
3. **Choose Agent**: Pick `beat-{module}-{agent}` mode
4. **Activate**: Mode is now active

### Mode Format

- Mode name: `beat-{module}-{agent}`
- Display: `{icon} {title}`
